The LEO Satellite Infrastructure Series

Four production-grade distributed systems, each solving a different failure mode of a low-Earth-orbit satellite network — task scheduling, anomaly response, offline-first sync, and live monitoring. They're built to interoperate, so they're presented here as one connected system rather than four separate projects.

🛰️
Ground Station Task Scheduler

GroundLink

Fault-tolerant distributed task scheduler using a priority-queue algorithm (O log n) across 4 ground stations. Zero task loss during failures, via real-time reassignment and heartbeat monitoring.

View on GitHub →
🤖
AI Satellite Operations Agent

KuiperOps

AI ops agent using Groq LLaMA3, detecting anomalies across 6 satellite failure types and auto-generating incident runbooks — reducing diagnosis time by 80% versus manual triage.

Remote Device Sync System

EdgeSync

Offline-first sync enabling remote devices (schools, hospitals) to queue 1,000+ records in SQLite during satellite outages and auto-sync to AWS DynamoDB on reconnect. Zero data loss.
